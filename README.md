*Deep competitive intelligence analysis with market positioning, feature comparison, and strategic insights.*

An AI-powered competitive intelligence platform that monitors and analyzes your competitors across every dimension. From pricing and product features to marketing strategy and customer sentiment, get actionable insights that help you stay ahead of the competition.

## Features

| Feature | Description |
|---------|-------------|
| Competitor Profiling | Builds comprehensive profiles of competitors including products, pricing, team, and funding |
| Feature Comparison Matrix | Creates detailed side-by-side feature comparisons with gap analysis |
| Market Positioning Map | Visualizes competitive landscape and identifies white space opportunities |
| Strategy Decoder | Analyzes competitor marketing, content, and go-to-market strategies |
| SWOT Generator | Auto-generates SWOT analysis based on competitive data |
| Alert System | Monitors competitor moves including pricing changes, product launches, and hiring trends |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-competitor-analyzer-skill
```

## Usage

```
profile [competitor]       # Build a comprehensive competitor profile
compare [us vs them]       # Side-by-side feature comparison
map market [industry]      # Generate competitive landscape map
swot [competitor]          # Auto-generate SWOT analysis
monitor [competitor list]  # Set up competitive monitoring alerts
```
